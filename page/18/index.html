SparkеӯҰд№ з¬”и®°вҖ”вҖ”Spark on YARN

2017-04-10
tonglin0325

SparkиҝҗиЎҢзҡ„ж—¶еҖҷпјҢйҮҮз”Ёзҡ„жҳҜдё»д»Һз»“жһ„пјҢжңүдёҖдёӘиҠӮзӮ№иҙҹиҙЈдёӯеӨ®еҚҸи°ғпјҢ и°ғеәҰеҗ„дёӘеҲҶеёғејҸе·ҘдҪңиҠӮзӮ№гҖӮиҝҷдёӘдёӯеӨ®еҚҸи°ғиҠӮзӮ№иў«з§°дёәй©ұеҠЁеҷЁпјҲ Driverпјү иҠӮзӮ№гҖӮдёҺд№ӢеҜ№еә”зҡ„е·ҘдҪңиҠӮзӮ№иў«з§°дёәжү§иЎҢеҷЁпјҲ executorпјү иҠӮзӮ№гҖӮ

жүҖжңүзҡ„ Spark зЁӢеәҸйғҪйҒөеҫӘеҗҢж ·зҡ„з»“жһ„пјҡзЁӢеәҸд»Һиҫ“е…Ҙж•°жҚ®еҲӣе»әдёҖзі»еҲ— RDDпјҢ еҶҚдҪҝз”ЁиҪ¬еҢ–ж“ҚдҪңжҙҫз”ҹеҮәж–°зҡ„ RDDпјҢжңҖеҗҺдҪҝз”ЁиЎҢеҠЁж“ҚдҪңж”¶йӣҶжҲ–еӯҳеӮЁз»“жһң RDD дёӯзҡ„ж•°жҚ®гҖӮ

1.й©ұеҠЁеҷЁиҠӮзӮ№пјҡ

Spark й©ұеҠЁеҷЁжҳҜжү§иЎҢдҪ зҡ„зЁӢеәҸдёӯзҡ„ main() ж–№жі•зҡ„иҝӣзЁӢгҖӮе®ғжү§иЎҢз”ЁжҲ·зј–еҶҷзҡ„з”ЁжқҘеҲӣе»ә SparkContextгҖҒеҲӣе»ә RDDпјҢд»ҘеҸҠиҝӣиЎҢ RDD зҡ„иҪ¬еҢ–ж“ҚдҪңе’ҢиЎҢеҠЁж“ҚдҪңзҡ„д»Јз ҒгҖӮе…¶е®һпјҢеҪ“дҪ еҗҜеҠЁ Spark shell ж—¶пјҢдҪ е°ұеҗҜеҠЁдәҶдёҖдёӘ Spark й©ұеҠЁеҷЁзЁӢеәҸ

й©ұеҠЁеҷЁзЁӢеәҸеңЁ Spark еә”з”ЁдёӯжңүдёӢиҝ°дёӨдёӘиҒҢиҙЈпјҡ1.жҠҠз”ЁжҲ·зЁӢеәҸиҪ¬дёәд»»еҠЎ 2.дёәжү§иЎҢеҷЁиҠӮзӮ№и°ғеәҰд»»еҠЎ

2.жү§иЎҢеҷЁиҠӮзӮ№пјҡ

Spark жү§иЎҢеҷЁиҠӮзӮ№жҳҜдёҖз§Қе·ҘдҪңиҝӣзЁӢпјҢиҙҹиҙЈеңЁ Spark дҪңдёҡдёӯиҝҗиЎҢд»»еҠЎпјҢд»»еҠЎй—ҙзӣёдә’зӢ¬з«ӢгҖӮ Spark еә”з”ЁеҗҜеҠЁж—¶пјҢ жү§иЎҢеҷЁиҠӮзӮ№е°ұиў«еҗҢж—¶еҗҜеҠЁпјҢе№¶дё”е§Ӣз»ҲдјҙйҡҸзқҖж•ҙдёӘ Spark еә”з”Ёзҡ„з”ҹе‘Ҫе‘ЁжңҹиҖҢеӯҳеңЁгҖӮ

жү§иЎҢеҷЁиҝӣзЁӢжңүдёӨеӨ§дҪңз”Ёпјҡ з¬¬дёҖпјҢе®ғд»¬иҙҹиҙЈиҝҗиЎҢз»„жҲҗ Spark еә”з”Ёзҡ„д»»еҠЎпјҢе№¶е°Ҷз»“жһңиҝ”еӣһз»ҷй©ұеҠЁеҷЁиҝӣзЁӢпјӣ з¬¬дәҢпјҢе®ғд»¬йҖҡиҝҮиҮӘиә«зҡ„еқ—з®ЎзҗҶеҷЁпјҲBlock Managerпјүдёәз”ЁжҲ·зЁӢеәҸдёӯиҰҒжұӮзј“еӯҳзҡ„ RDD жҸҗдҫӣеҶ…еӯҳејҸеӯҳеӮЁгҖӮ

3.йӣҶзҫӨз®ЎзҗҶеҷЁпјҡ

й©ұеҠЁеҷЁиҠӮзӮ№е’Ңжү§иЎҢеҷЁиҠӮзӮ№жҳҜеҰӮдҪ•еҗҜеҠЁзҡ„е‘ўпјҹ Spark дҫқиө–дәҺйӣҶзҫӨз®ЎзҗҶеҷЁжқҘеҗҜеҠЁжү§иЎҢеҷЁиҠӮзӮ№пјҢиҖҢеңЁжҹҗдәӣзү№ж®Ҡжғ…еҶөдёӢпјҢд№ҹдҫқиө–йӣҶзҫӨз®ЎзҗҶеҷЁжқҘеҗҜеҠЁй©ұеҠЁеҷЁиҠӮзӮ№гҖӮ
