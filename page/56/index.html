1.ж•°жҚ®еҲҶеұӮ#

2.ж•°д»“е»әжЁЎж–№жі•#

еңЁж•°жҚ®д»“еә“жЁЎеһӢдёӯпјҢжҳҹеһӢжЁЎеһӢе’ҢйӣӘиҠұеһӢжЁЎеһӢжҳҜдёӨдёӘеёёз”Ёзҡ„и®ҫи®ЎжЁЎејҸгҖӮеҸӮиҖғпјҡж•°жҚ®д»“еә“зі»еҲ—пјҡжҳҹеһӢжЁЎеһӢе’ҢйӣӘиҠұеһӢжЁЎеһӢ

1.жҳҹеһӢжЁЎеһӢ#

жҳҹеһӢжЁЎеһӢжҳҜдёҖз§Қз®ҖеҚ•зҡ„ж•°жҚ®д»“еә“жЁЎеһӢпјҢд№ҹжҳҜжңҖеёёи§Ғзҡ„жЁЎеһӢд№ӢдёҖгҖӮеңЁжҳҹеһӢжЁЎеһӢдёӯпјҢдёӯеҝғиЎЁпјҲз§°дёәдёҡеҠЎдәӢе®һиЎЁпјүиҝһжҺҘеҲ°еҮ дёӘз»ҙеәҰиЎЁпјҲз§°дёәдёҡеҠЎз»ҙеәҰиЎЁпјүгҖӮз»ҙеәҰиЎЁдёӯеҢ…еҗ«дәҶдёҡеҠЎзҡ„еҗ„дёӘзү№еҫҒпјҢеҰӮж—¶й—ҙгҖҒеҢәеҹҹгҖҒдә§е“ҒзӯүгҖӮ

еңЁ SQL дёӯпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёд»ҘдёӢиҜӯеҸҘжқҘеҲӣе»әдёҖдёӘжҳҹеһӢжЁЎеһӢпјҡ

2.йӣӘиҠұеһӢжЁЎеһӢ#

йӣӘиҠұеһӢжЁЎеһӢжҳҜеңЁжҳҹеһӢжЁЎеһӢеҹәзЎҖдёҠзҡ„жү©еұ•пјҢеӣ е…¶еҪўдјјйӣӘиҠұиҖҢеҫ—еҗҚгҖӮиҝҷз§ҚжЁЎеһӢеңЁжҳҹеһӢжЁЎеһӢзҡ„еҹәзЎҖдёҠпјҢе°Ҷз»ҙеәҰиЎЁжӢҶеҲҶжҲҗжӣҙе°Ҹзҡ„иЎЁеҪўејҸпјҢеҪўжҲҗеӨҡеұӮиЎЁзҡ„з»“жһ„гҖӮ

йӣӘиҠұеһӢжЁЎеһӢпјҡеҪ“жңүдёҖдёӘжҲ–еӨҡдёӘз»ҙиЎЁжІЎжңүзӣҙжҺҘиҝһжҺҘеҲ°дәӢе®һиЎЁдёҠпјҢиҖҢжҳҜйҖҡиҝҮе…¶д»–з»ҙиЎЁиҝһжҺҘеҲ°дәӢе®һиЎЁдёҠж—¶пјҢе…¶еӣҫи§Је°ұеғҸеӨҡдёӘйӣӘиҠұиҝһжҺҘеңЁдёҖиө·пјҢж•…з§°йӣӘиҠұжЁЎеһӢгҖӮйӣӘиҠұжЁЎеһӢжҳҜеҜ№жҳҹеһӢжЁЎеһӢзҡ„жү©еұ•гҖӮе®ғеҜ№жҳҹеһӢжЁЎеһӢзҡ„з»ҙиЎЁиҝӣдёҖжӯҘеұӮж¬ЎеҢ–пјҢеҺҹжңүзҡ„еҗ„з»ҙиЎЁеҸҜиғҪиў«жү©еұ•дёәе°Ҹзҡ„дәӢе®һиЎЁпјҢеҪўжҲҗдёҖдәӣеұҖйғЁзҡ„ вҖңеұӮж¬Ў вҖң еҢәеҹҹпјҢиҝҷдәӣиў«еҲҶи§Јзҡ„иЎЁйғҪиҝһжҺҘеҲ°дё»з»ҙеәҰиЎЁиҖҢдёҚжҳҜдәӢе®һиЎЁгҖӮ

йӣӘиҠұеһӢжЁЎеһӢйҖҡиҝҮжңҖеӨ§йҷҗеәҰең°еҮҸе°‘ж•°жҚ®еӯҳеӮЁйҮҸд»ҘеҸҠиҒ”еҗҲиҫғе°Ҹзҡ„з»ҙиЎЁжқҘж”№е–„жҹҘиҜўжҖ§иғҪгҖӮйӣӘиҠұеһӢз»“жһ„еҺ»йҷӨдәҶж•°жҚ®еҶ—дҪҷгҖӮ

жҳҹеһӢжЁЎеһӢеҜ№жҜ”йӣӘиҠұжЁЎеһӢ#

1гҖҒжҹҘиҜўжҖ§иғҪи§’еәҰжқҘзңӢ

еңЁOLAPпјҲHiveпјүе»әи®®з”ЁжҳҹеһӢжЁЎеһӢпјҢHiveиЎЁйҖҡиҝҮеӨ§йҮҸзҡ„еҶ—дҪҷжқҘжҸҗеҚҮжҹҘиҜўж•ҲзҺҮпјҢжҳҹеһӢжЁЎеһӢеҜ№OLAPзҡ„еҲҶжһҗеј•ж“Һж”ҜжҢҒжҜ”иҫғеҸӢеҘҪпјӣиҖҢйӣӘиҠұеһӢиҰҒеҒҡеӨҡдёӘиЎЁиҒ”жҺҘпјҢжҖ§иғҪдјҡдҪҺдәҺжҳҹеһӢжһ¶жһ„

еңЁOLTPпјҲMySQLпјүе»әи®®з”ЁйӣӘиҠұжЁЎеһӢпјҢз”ұдәҺйӣӘиҠұеһӢжһ¶жһ„жӣҙжңүеҲ©дәҺеәҰйҮҸеҖјзҡ„иҒҡеҗҲпјҢеӣ жӯӨжҖ§иғҪиҰҒй«ҳдәҺжҳҹеһӢжһ¶жһ„пјҢжүҖд»ҘйӣӘиҠұжЁЎеһӢеңЁе…ізі»еһӢж•°жҚ®еә“дёӯеҰӮMySQLпјҢOracleдёӯйқһеёёеёёи§Ғ

2гҖҒжЁЎеһӢеӨҚжқӮеәҰи§’еәҰ

жҳҹеһӢжһ¶жһ„жӣҙз®ҖеҚ•ж–№дҫҝеӨ„зҗҶ

3гҖҒеұӮж¬Ўз»“жһ„и§’еәҰ

йӣӘиҠұеһӢжһ¶жһ„жӣҙеҠ иҙҙиҝ‘OLTPзі»з»ҹзҡ„з»“жһ„пјҢжҜ”иҫғз¬ҰеҗҲдёҡеҠЎйҖ»иҫ‘пјҢеұӮж¬ЎжҜ”иҫғжё…жҷ°гҖӮ

4гҖҒеӯҳеӮЁи§’еәҰ

йӣӘиҠұеһӢжһ¶жһ„е…·жңүе…ізі»ж•°жҚ®жЁЎеһӢзҡ„жүҖжңүдјҳзӮ№пјҢдёҚдјҡдә§з”ҹеҶ—дҪҷж•°жҚ®пјҢиҖҢзӣёжҜ”д№ӢдёӢжҳҹеһӢжһ¶жһ„дјҡдә§з”ҹж•°жҚ®еҶ—дҪҷгҖӮ

3.Data Vault жЁЎеһӢ#

Data Vault жҳҜеҸҰдёҖз§Қж•°жҚ®д»“еә“е»әжЁЎж–№жі•пјҢжҳҜ Dan Linstedt еңЁ 20 дё–зәӘ 90 е№ҙд»ЈжҸҗеҮәзҡ„пјҢдё»иҰҒз”ЁдәҺдјҒдёҡзә§зҡ„ж•°жҚ®д»“еә“е»әжЁЎгҖӮ

Data Vault жЁЎеһӢз”ұдёӯеҝғиЎЁпјҲHubпјүгҖҒй“ҫжҺҘиЎЁпјҲLinkпјүгҖҒйҷ„еұһиЎЁпјҲSatelliteпјүдёүдёӘдё»иҰҒз»„жҲҗйғЁеҲҶгҖӮе…¶дёӯпјҢдёӯеҝғиЎЁжҳҜж ёеҝғпјҢз”ЁдәҺеӯҳеӮЁдёҡеҠЎдё»й”®пјҢй“ҫжҺҘиЎЁи®°еҪ•дёҡеҠЎе…ізі»пјҢйҷ„еұһиЎЁи®°еҪ•дёҡеҠЎжҸҸиҝ°гҖӮ

пјҲ1пјүдёӯеҝғиЎЁ

дёӯеҝғиЎЁз”ЁжқҘеӯҳеӮЁдјҒдёҡжҜҸдёӘдёҡеҠЎе®һдҪ“зҡ„дёҡеҠЎдё»й”®пјҢдёҡеҠЎдё»й”®е”ҜдёҖж ҮиҜҶжҹҗдёӘдёҡеҠЎе®һдҪ“гҖӮдёӯеҝғиЎЁе’Ңжәҗзі»з»ҹжҳҜзӣёдә’зӢ¬з«Ӣзҡ„пјҢеҚіж— и®әдёҡеҠЎдё»й”®жҳҜеҗҰз”ЁдәҺеӨҡдёӘдёҡеҠЎзі»з»ҹпјҢе®ғеңЁ Data Vault дёӯеҸӘдҝқз•ҷдёҖд»ҪпјҢе…¶д»–зҡ„з»„д»¶йғҪй“ҫжҺҘеҲ°иҝҷдёҖдёӘдёҡеҠЎдё»й”®дёҠгҖӮ

еҮәдәҺи®ҫи®ЎдёҠзҡ„иҖғиҷ‘пјҢдёӯеҝғиЎЁдёҖиҲ¬з”ұдё»й”®гҖҒдёҡеҠЎдё»й”®гҖҒиЈ…иҪҪж—¶й—ҙжҲігҖҒж•°жҚ®жқҘжәҗзі»з»ҹеӣӣдёӘеӯ—ж®өз»„жҲҗгҖӮе…¶дёӯдё»й”®жҳҜзі»з»ҹз”ҹжҲҗзҡ„д»ЈзҗҶй”®пјҢд»…дҫӣеҶ…йғЁдҪҝз”ЁгҖӮ

пјҲ2пјүй“ҫжҺҘиЎЁ

й“ҫжҺҘиЎЁжҳҜдёҚеҗҢдёӯеҝғиЎЁзҡ„й“ҫжҺҘгҖӮдёҖдёӘй“ҫжҺҘиЎЁдёҖиҲ¬еңЁдёӨдёӘжҲ–еӨҡдёӘдёӯеҝғиЎЁд№Ӣй—ҙжңүе…іиҒ”гҖӮдёҖдёӘй“ҫжҺҘиЎЁйҖҡеёёжҳҜдёҖдёӘеӨ–й”®пјҢиЎЁзӨәдёҖз§ҚдёҡеҠЎе…ізі»пјҢжҜ”еҰӮпјҡдәӨжҳ“иЎЁгҖҒе®ўжҲ·е…іиҒ”иҙҰжҲ·зӯүгҖӮ

й“ҫжҺҘиЎЁдё»иҰҒеҢ…жӢ¬дё»й”®гҖҒеӨ–й”®1гҖҒ……гҖҒеӨ–й”®nгҖҒиЈ…иҪҪж—¶й—ҙжҲігҖҒж•°жҚ®жқҘжәҗзі»з»ҹзӯүеӯ—ж®өжһ„жҲҗпјҢе…¶дёӯдё»й”®еҜ№еә”еӨҡдёӘеӨ–й”®зҡ„е”ҜдёҖз»„еҗҲпјҢдёҖиҲ¬жҳҜдёҺдёҡеҠЎж— е…ізҡ„еәҸеҲ—ж•°еҖјгҖӮ

пјҲ3пјүйҷ„еұһиЎЁ

йҷ„еұһиЎЁз”ЁжқҘдҝқеӯҳдёӯеҝғиЎЁе’Ңй“ҫжҺҘиЎЁзҡ„жҸҸиҝ°еұһжҖ§пјҢеҢ…еҗ«жүҖжңүеҺҶеҸІеҸҳеҢ–ж•°жҚ®пјҢйҷ„еұһиЎЁжңүдё”д»…жңүдёҖдёӘе”ҜдёҖеӨ–й”®е…іиҒ”еҲ°дёӯеҝғиЎЁжҲ–й“ҫжҺҘиЎЁгҖӮ

йҷ„еұһиЎЁдё»иҰҒеҢ…жӢ¬дё»й”®гҖҒеӨ–й”®гҖҒеұһжҖ§1гҖҒ……гҖҒеұһжҖ§nгҖҒиЈ…иҪҪж—¶й—ҙгҖҒеӨұж•Ҳж—¶й—ҙгҖҒж•°жҚ®жқҘжәҗзі»з»ҹпјҢдё»й”®з”ЁдәҺе”ҜдёҖж ҮиҜҶйҷ„еұһиЎЁдёӯзҡ„дёҖиЎҢи®°еҪ•пјҢдёҖиҲ¬жҳҜдёҺдёҡеҠЎж— е…ізҡ„еәҸеҲ—ж•°еҖјгҖӮ

йқһиҒҡз°Үзҙўеј•зҡ„и§ЈйҮҠжҳҜпјҡзҙўеј•йЎәеәҸдёҺж•°жҚ®зү©зҗҶжҺ’еҲ—йЎәеәҸж— е…і

 

е…·дҪ“еҸӮиҖғпјҡж·ұе…ҘзҗҶи§ЈMySQLзҙўеј•еҺҹзҗҶе’Ңе®һзҺ°——дёәд»Җд№Ҳзҙўеј•еҸҜд»ҘеҠ йҖҹжҹҘиҜўпјҹ

 

дёҠеӣҫдёӯзҡ„Secondary Keyе°ұжҳҜдәҢзә§зҙўеј•

еҸӮиҖғпјҡMysqlиҒҡз°Үзҙўеј•е’ҢдәҢзә§зҙўеј•еҲ°еә•жңүдҪ•дёҚеҗҢ

еӯҳеӮЁеҢәеҲ«

InnodbеңЁзЈҒзӣҳдёҠдјҡеӯҳеӮЁ2дёӘж–Үд»¶пјҡfrmж–Үд»¶еӯҳеӮЁиЎЁе®ҡд№үпјҢidbеӯҳеӮЁж•°жҚ®е’Ңзҙўеј•ж–Үд»¶

гҖҖгҖҖ

й”ҒеҢәеҲ«

MyISAMеӯҳеӮЁеј•ж“ҺдҪҝз”Ёзҡ„жҳҜиЎЁй”ҒпјҢInnodbеӯҳеӮЁеј•ж“Һй»ҳи®ӨдҪҝз”Ёзҡ„жҳҜиЎҢй”ҒпјҢеҗҢж—¶д№ҹж”ҜжҢҒиЎЁзә§й”Ғ

еҸӮиҖғпјҡгҖҗдёҖеҲҶй’ҹзі»еҲ—гҖ‘дёҖеҲҶй’ҹеҝ«йҖҹдәҶи§ЈMyISAMе’ҢInnodbзҡ„дё»иҰҒеҢәеҲ«

 

Bж ‘

BTreeжҳҜе№іиЎЎжҗңзҙўеӨҡеҸүж ‘пјҢи®ҫж ‘зҡ„еәҰдёә2dпјҲd>1пјүпјҢй«ҳеәҰдёәhпјҢйӮЈд№ҲBTreeиҰҒж»Ўи¶ід»ҘдёҖдёӢжқЎд»¶пјҡ

жҜҸдёӘеҸ¶еӯҗз»“зӮ№зҡ„й«ҳеәҰдёҖж ·пјҢзӯүдәҺhпјӣ

жҜҸдёӘйқһеҸ¶еӯҗз»“зӮ№з”ұn-1дёӘkeyе’ҢnдёӘжҢҮй’Ҳpointз»„жҲҗпјҢе…¶дёӯd<=n<=2d,keyе’Ңpointзӣёдә’й—ҙйҡ”пјҢз»“зӮ№дёӨз«ҜдёҖе®ҡжҳҜkeyпјӣ

еҸ¶еӯҗз»“зӮ№жҢҮй’ҲйғҪдёәnullпјӣ

йқһеҸ¶еӯҗз»“зӮ№зҡ„keyйғҪжҳҜ[key,data]дәҢе…ғз»„пјҢе…¶дёӯkeyиЎЁзӨәдҪңдёәзҙўеј•зҡ„й”®пјҢdataдёәй”®еҖјжүҖеңЁиЎҢзҡ„ж•°жҚ®пјӣ

 

еңЁBTreeзҡ„з»“жһ„дёӢпјҢе°ұеҸҜд»ҘдҪҝз”ЁдәҢеҲҶжҹҘжүҫзҡ„жҹҘжүҫж–№ејҸпјҢжҹҘжүҫеӨҚжқӮеәҰдёәh*log(n)пјҢдёҖиҲ¬жқҘиҜҙж ‘зҡ„й«ҳеәҰжҳҜеҫҲе°Ҹзҡ„пјҢдёҖиҲ¬дёә3е·ҰеҸіпјҢеӣ жӯӨBTreeжҳҜдёҖдёӘйқһеёёй«ҳж•Ҳзҡ„жҹҘжүҫз»“жһ„гҖӮ

 

B+ж ‘

B+TreeжҳҜBTreeзҡ„дёҖдёӘеҸҳз§ҚпјҢи®ҫdдёәж ‘зҡ„еәҰж•°пјҢhдёәж ‘зҡ„й«ҳеәҰпјҢB+Treeе’ҢBTreeзҡ„дёҚеҗҢдё»иҰҒеңЁдәҺпјҡ

B+Treeдёӯзҡ„йқһеҸ¶еӯҗз»“зӮ№дёҚеӯҳеӮЁж•°жҚ®пјҢеҸӘеӯҳеӮЁй”®еҖјпјӣ

B+Treeзҡ„еҸ¶еӯҗз»“зӮ№жІЎжңүжҢҮй’ҲпјҢжүҖжңүй”®еҖјйғҪдјҡеҮәзҺ°еңЁеҸ¶еӯҗз»“зӮ№дёҠпјҢдё”keyеӯҳеӮЁзҡ„й”®еҖјеҜ№еә”dataж•°жҚ®зҡ„зү©зҗҶең°еқҖпјӣ

B+Treeзҡ„жҜҸдёӘйқһеҸ¶еӯҗиҠӮзӮ№з”ұnдёӘй”®еҖјkeyе’ҢnдёӘжҢҮй’Ҳpointз»„жҲҗпјӣ

 

B+TreeеҜ№жҜ”BTreeзҡ„дјҳзӮ№#

1гҖҒзЈҒзӣҳиҜ»еҶҷд»Јд»·жӣҙдҪҺ

дёҖиҲ¬жқҘиҜҙB+TreeжҜ”BTreeжӣҙйҖӮеҗҲе®һзҺ°еӨ–еӯҳзҡ„зҙўеј•з»“жһ„пјҢеӣ дёәеӯҳеӮЁеј•ж“Һзҡ„и®ҫи®Ўдё“е®¶е·§еҰҷзҡ„еҲ©з”ЁдәҶеӨ–еӯҳпјҲзЈҒзӣҳпјүзҡ„еӯҳеӮЁз»“жһ„пјҢеҚізЈҒзӣҳзҡ„жңҖе°ҸеӯҳеӮЁеҚ•дҪҚжҳҜжүҮеҢәпјҲsectorпјүпјҢиҖҢж“ҚдҪңзі»з»ҹзҡ„еқ—пјҲblockпјүйҖҡеёёжҳҜж•ҙж•°еҖҚзҡ„sectorпјҢж“ҚдҪңзі»з»ҹд»ҘйЎөпјҲpageпјүдёәеҚ•дҪҚз®ЎзҗҶеҶ…еӯҳпјҢдёҖйЎөпјҲpageпјүйҖҡеёёй»ҳи®Өдёә4KпјҢж•°жҚ®еә“зҡ„йЎөйҖҡеёёи®ҫзҪ®дёәж“ҚдҪңзі»з»ҹйЎөзҡ„ж•ҙж•°еҖҚпјҢеӣ жӯӨзҙўеј•з»“жһ„зҡ„иҠӮзӮ№иў«и®ҫи®ЎдёәдёҖдёӘйЎөзҡ„еӨ§е°ҸпјҢз„¶еҗҺеҲ©з”ЁеӨ–еӯҳзҡ„“йў„иҜ»еҸ–”еҺҹеҲҷпјҢжҜҸж¬ЎиҜ»еҸ–зҡ„ж—¶еҖҷпјҢжҠҠж•ҙдёӘиҠӮзӮ№зҡ„ж•°жҚ®иҜ»еҸ–еҲ°еҶ…еӯҳдёӯпјҢз„¶еҗҺеңЁеҶ…еӯҳдёӯжҹҘжүҫпјҢе·ІзҹҘеҶ…еӯҳзҡ„иҜ»еҸ–йҖҹеәҰжҳҜеӨ–еӯҳиҜ»еҸ–I/OйҖҹеәҰзҡ„еҮ зҷҫеҖҚпјҢйӮЈд№ҲжҸҗеҚҮжҹҘжүҫйҖҹеәҰзҡ„е…ій”®е°ұеңЁдәҺе°ҪеҸҜиғҪе°‘зҡ„зЈҒзӣҳI/OпјҢйӮЈд№ҲеҸҜд»ҘзҹҘйҒ“пјҢжҜҸдёӘиҠӮзӮ№дёӯзҡ„keyдёӘж•°и¶ҠеӨҡпјҢйӮЈд№Ҳж ‘зҡ„й«ҳеәҰи¶Ҡе°ҸпјҢйңҖиҰҒI/Oзҡ„ж¬Ўж•°и¶Ҡе°‘пјҢеӣ жӯӨдёҖиҲ¬жқҘиҜҙB+TreeжҜ”BTreeжӣҙеҝ«пјҢеӣ дёәB+Treeзҡ„йқһеҸ¶иҠӮзӮ№дёӯдёҚеӯҳеӮЁdataпјҢе°ұеҸҜд»ҘеӯҳеӮЁжӣҙеӨҡзҡ„keyгҖӮ

2гҖҒжҹҘиҜўйҖҹеәҰжӣҙзЁіе®ҡ
